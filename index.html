📘 Sistema de Evaluación y Plan de Estudio Personalizado

Flujo automatizado con DeepSeek API · Evaluaciones semanales · Informes LaTeX

→ fortalezas/debilidades] Eval1 --> GuardaHist[Guardar en progreso.json] GuardaHist --> InformeSem[Generar informe semanal LaTeX
para padres y tutor] InformeSem --> Viernes{¿Es viernes?} Viernes -->|No| Espera[Esperar hasta viernes] Espera --> Viernes Viernes -->|Sí| Secretaria[Secretaria introduce
progreso.json actual] Secretaria --> CheckSemana{¿Semana actual >= 8?} CheckSemana -->|No| Siguiente[Semana = semana+1

preguntas mezcladas] GenExamen --> EstudianteN[Estudiante responde] EstudianteN --> EvalN[Evaluar resultados + comparar] EvalN --> ActualizaJson[Actualizar progreso.json] ActualizaJson --> InformeSem CheckSemana -->|Sí| FinSemana8[Completadas 8 semanas] FinSemana8 --> InformeFinal[DeepSeek genera informe
antes/despues por estudiante] InformeFinal --> LatexFinal[Compilar informes LaTeX → PDF] LatexFinal --> Fin([Fin del proceso]) Error --> Fin
🎨 Leyenda de colores
Inicio / Fin    Decisión    Proceso / Acción    Base de datos / JSON    Interacción con DeepSeek API    Generación de informes LaTeX
⚙️ Mejora clave: El sistema guarda automáticamente cada evaluación semanal. La secretaria solo debe cargar el JSON cada viernes.
🔁 Adaptable a múltiples universidades mediante archivo universidades_config.json.
